## ⚙ 설치 및 실행

```bash
# 프로젝트 클론
git clone https://github.com/VaIice/PDF-Parser.git

# 프로젝트 디렉토리로 이동
cd PDF-Parser

# frontend 디렉토리로 이동
cd frontend

# 필요한 의존성 설치
npm install

# backend 디렉토리로 이동
cd ../backend

# 필요한 의존성 설치
npm install

# backend 폴더에서 프론트엔드와 백엔드를 동시에 실행 (터미널 나눌 필요 X)
npm run dev
```

#### 개발 서버 실행 후 localhost:5173 접속

---

## 🔍 구현 사항

- PDF 파일 업로드
- 웹 애플리케이션 내에서 뷰어로 표시
- 텍스트 추출 기능 구현
- 데이터의 특정 패턴을 분석하여 필요한 정보 추출 및 처리
- 테이블 형식 시각화
- txt 데이터 다운로드
